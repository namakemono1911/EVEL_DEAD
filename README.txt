//-------------------------------
//はじめに
//-------------------------------
この作品は4年次に6カ月の期間で制作した一人称視点のゲームです。

//-------------------------------
//更新履歴
//-------------------------------
現在進行形でアップデート中

//-------------------------------
//操作方法
//-------------------------------
視点移動：マウス
移動：WASD
構える：右クリック
発射：右クリックで構えながら左クリック
画面遷移：スペースキー

//-------------------------------
//頑張ったところ
//-------------------------------
このゲームを制作するにあたり、マップエディタ、モデルエディタ、モーションエディタ、コリジョンエディタ、を制作し
そのツールを使い制作しました。
またモーションしている物体との当たり判定処理や壁に弾痕を残す処理には特に力を入れて制作しました。
